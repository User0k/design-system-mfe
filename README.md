## Components

### Core Components

- **Button** - Multiple variants (primary, secondary, success, warning, error) and sizes
- **Input** - Form inputs with validation states, helper text, and multiple styles
- **Radio** - Accessible radio button groups
- **Checkbox** - Checkbox inputs with indeterminate state support
- **Typography** - Consistent text hierarchy (H1-H6, body, caption, overline)

## Accessibility

All components are built with accessibility in mind:

- Proper ARIA attributes
- Keyboard navigation support
- Screen reader compatibility
- Focus management
- Color contrast compliance
